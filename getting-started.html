危险

Adventure 文档目前仍为 尚在施工 状态，并且作为 Javadocs 的补充. 一些地方可能覆盖面有限，或可能不是完全最新的. 所有你有任何问题，欢迎加入我们的 Discord.

开始

要想在你的项目中使用 Adventure, 你需要添加如下的依赖和仓库 (如果使用 Gradle):

 <dependency>
    <groupId>net.kyori</groupId>
    <artifactId>adventure-api</artifactId>
    <version>4.9.3</version>
 </dependency>

 repositories {
   mavenCentral()
 }

 dependencies {
    implementation "net.kyori:adventure-api:4.9.3"
 }

 repositories {
   mavenCentral()
 }

 dependencies {
    implementation("net.kyori:adventure-api:4.9.3")
 }

一些平台已经原生支持 Adventure. 在这种情况下, 你无需添加 Adventure 作为依赖. 要想查看已经包含 Adventure 的平台列表, 请见 原生支持.

要想在其他平台使用 Adventure, 你可能希望查看一些平台特定 (platform-specific) 的适配器 (adapters). 一个带有受支持适配器的平台列表可见于 平台.

使用快照构建

要想使用快照构建, 你需要添加如下的仓库:

<repositories>
    <repository>
       <id>sonatype-oss-snapshots</id>
       <url>https://oss.sonatype.org/content/repositories/snapshots/</url>
    </repository>
</repositories>

repositories {
   maven {
       name = "sonatype-oss-snapshots"
       url = "https://oss.sonatype.org/content/repositories/snapshots/"
   }
}

repositories {
   maven(url = "https://oss.sonatype.org/content/repositories/snapshots/") {
       name = "sonatype-oss-snapshots"
   }
}
