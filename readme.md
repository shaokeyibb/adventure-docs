# adventure-docs

adventure 的 [文档](https://docs.adventure.kyori.net/) 的源码.

## 使用方法

1. 推荐使用 virtualenv:
   1. 安装: `apt install virtualenv` (或者在你的操作系统上使用其他的形式安装)
   1. `virtualenv ENV`
   1. `. ENV/bin/activate`  
1. 安装所有必须的依赖库: `pip install -r requirements.txt`
1. 运行开发服务器 `make livehtml`
1. 做你想做的事!
